# School Forum Prototype

Прототип школьного форума с React, Express, SQLite и Prisma.

## Быстрый старт

```bash
npm run setup
npm run dev
```

По умолчанию:

- Frontend: http://localhost:5173
- API: http://localhost:4000

Seed создает модератора:

- Никнейм: `moderator`
- Пароль: `moderator123`

## Возможности

- Регистрация и вход по уникальному никнейму.
- JWT-сессия в HTTP-only cookie.
- Разделы и подразделы школьного форума.
- Создание постов только для авторизованных пользователей.
- Лайки, комментарии и простая вложенность комментариев.
- Модерация постов и комментариев пользователем с ролью `MODERATOR`.
